# SlimAdapter
SlimAdapter is a tiny library for fast creating adapters for RecycleView

## Requirements:
Android api 4.3+:

Add to dependencies:
```
implementation 'com.github.gerp83:SlimAdapter:1.05'
```

## Examples
Very good example can be found in the demo folder.
